Dropdown Menu

Working

This requires no controller as such. It's a pure directive & adds removes following styles on mouse movements:

  1. is-active to li
  2. js-dropdown-activeto ul

You can further customize this dropdown by adding these attributes to dropdown-menu attribute:

  1. disable-hover="true"
  2. disable-click-open="true"
  3. closing-time="500" in milli seconds

Modal

It needs a controller & template to work. Template can be added in html itself. Controller contains a single function called open(options). open(options) returns an ModelInstance object which can be closed, dismissed, repositioned etc.

  • Customize
    • size Controls size of model: tiny small large fullscreen
    • backdrop Controls presence of backdrop
  • Ways to close
    • closeOnClick determines whether clicking model will close the modal or not
    • keyboardESC to close
  • Give template
    • templateUrl will give the tempate url location
    • template will take in an inline template like "Hello World"
  • Declare Scope
    • scope We can use this to declare the scope we want to use. Defaults to $rootScope ie. $modal will create a new childscope as $scope from $rootScope and thus will not use scope of the parent controller.
    • resolve Passes the current controller's $scope as locals for $modal to use.
  • Declare Controller
    • controller
    • controllerAs
