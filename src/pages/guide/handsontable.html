Handsontable Django

Pre-Requisites

This requries Django & ExpressJS to run on the same host

  • Proxy ExpressJS at :3011 using Nginx
  • Proxy Django :8000/api to :3011/api using Nginx
  • Check :3011/api/v1/users/ is working or not
For local-development
Use best practice django from this link. Create the required virtualenv & run it. It will use the user model here.
  • Ensure you are running the page on 3011 port
  • Ensure that django is running
 For production instance For production instance check whether this link is working or not.
